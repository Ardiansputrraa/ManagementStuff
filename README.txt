File ini menjelaskan kelengkapan berkas proyek akhir kami yang berjudul Aplikasi Management Stuff.
Berkas terdiri dari:
dalam src terdapat package GUI, Img, Interface, Koneksi, main, Sound
File database sql dengan nama database: dataperusahaan
jika ingin menjalankan program harus memasukan JAR JCalender dan mysql-connector
Source code project NetBeans

Program ini akan membantu para small bisnis untuk mengatur permorma bisnis,
dan memberikan pengamanan dari para karyawan yang mencoba untuk berbuat curang,
sehingga karyawan tidak dapat merubah data dan harga yang ada pada database

Fitur-fitur Aplikasi:
1. Log In (Owner, Karyawan)
2. Log Out (Owner, Karyawan)
3. Menu Navigation (Dashboard, Data Produk, Data Karyawan, Akun & Absensi)
4. Table (Data Produk, Data Karyawan, Akun Karyawan, Absensi Karyawan)
5. Kode Items
6. Nama Items
7. Harga Items
8. Stock Items
9. Search
10. Refresh
11. NIK
12. Date and Time
13. Email
14. Nama Lengkap
15. Jabatan
16. Nomor Telepon
17. Radio Button (Pria, Wanita)
18. Tambah (Data Produk, Data Karyawan)
19. Edit (Data Produk, Data Karyawan)
20. Hapus (Data Produk, Data Karyawan)
21. Isi Absensi
22. Check Box (Hadir)
23. Pilih Log In
24. Option Absen
